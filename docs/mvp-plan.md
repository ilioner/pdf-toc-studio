# MVP Plan

## Phase 1: Core Engine

- Build TOC text parser
- Build logical-to-physical mapping
- Write bookmarks to PDF
- Write page labels to PDF
- Add validation checks

## Phase 2: Desktop UI

- File picker for PDF and TOC
- Text area for TOC input
- Offset input
- Preview panel
- Export action

## Phase 3: Packaging

- Tauri desktop wrapper
- Test on macOS
- Test on Windows
- Test on Linux

## Definition of Done

- User can import a PDF and TOC text
- User can set offset
- Exported PDF opens with correct bookmarks
- Exported PDF shows correct logical page labels
- Workflow works on at least one sample textbook
