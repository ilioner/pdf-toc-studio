# Roadmap

## Stage 1: Working MVP

Status: mostly complete

- shared Python TOC engine
- CLI preview and export
- Tk desktop app
- Tauri scaffold
- bridge contract between frontend and Python core

## Stage 2: Usability

- import TOC from file
- drag and drop PDF
- editable TOC tree
- better validation messages
- recent files
- project save/load

## Stage 3: Verification

- embedded PDF preview
- jump-to-page validation
- page label inspector
- sample datasets and regression tests

## Stage 4: Packaging

- PyInstaller bundle for Tk app
- working Tauri dev build
- signed macOS package
- Windows installer
- Linux AppImage or deb package

## Stage 5: Advanced Features

- OCR-assisted TOC extraction
- printed page number detection
- multi-scheme page labels
- batch processing
- bilingual UI
